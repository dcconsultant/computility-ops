# 服务器故障率特性高级功能方案（V2）

## 已确认口径（老板确认）

1. 主展示分子：故障次数
2. 同一台服务器同一记录年多次故障：按多次计数
3. `warranty_end_date` 为空：默认排除
4. 年龄桶上限：`1-8, 8+`

## 记录年定义

- 第1记录年：2021-04-07 ~ 2022-04-06
- 之后每年4月7日开始一个新记录年

## 分母规则

- 某服务器在某记录年纳入分母条件：
  - `warranty_end_date >= 该记录年第一天`
- 机龄：记录年第一天与投产日期 `launch_date` 的年份差，映射到 `1-8,8+`
- 权重：
  - 温/热存储：`1 + data_disk_count`
  - 其他：`1`

## 分子规则

- 分子为故障次数
- 故障按事件时间归属到对应记录年
- 故障事件对应机龄按“记录年第一天”的机龄桶计算

## 聚合维度

支持单独或组合汇总：
- 环境范围：全量、生产、开测
- 场景大类：存储、非存储、计算、温存储、热存储、GPU
- 配置类型
- 型号（厂商+型号）

## 输出建议

- 记录年 x 机龄桶 的故障次数、分母权重、故障率
- 支持 drill-down（范围→场景→配置→型号）
